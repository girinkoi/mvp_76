Aplikasi MVP 76 Jual Beli Tanaman Hias

-----------------------------

Fitur:

Login user

Manajemen 

Sistem role (admin & user)

Autentikasi dari database

-----------------------------

Di admin ada 4 page:

page 1 (dashboard):
{
    admin bisa melihat barang apa saja yang sedang di jual. Yang ready, pending (menunggu konfirmasi), dan sold (terjual).
}


page 2 (crud jual):
{
    tempat admin bisa menjual barang.
}



page 3 (pengelolaan):
{
    admin bisa melihat, barang apa yang sudah di beli, disini admin bisa melihat tanaman yang dibeli, nama pembeli, harga, alamat, dan metode pembayaran, disini admin juga bisa mengkonfirmasi barang, apakah barang sudah sampai tangan konsumen atau belum, jika sudah, admin bisa menekan tombol "Accept", dan status barang jadi completed/sold.
}

page 4 (History):
{
    admin bisa melihat history barang yang sudah terjual
}
